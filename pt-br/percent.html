sobre

A unidade de porcentagem no CSS é relativa ao bloco onde o elemento está contido. É ótimo para imagens: aqui mostramos uma imagem que sempre terá 50% da largura de seu container. Redimensione a janela do seu navegador para ver o que acontece!

<article class="clearfix">

Você também poderia utilizar min-width e max-width para limitar o máximo e o mínimo de largura que a imagem poderia ter.

</article>

layout com porcentagem de largura

Você pode utilizar porcentagem para estruturar uma página, mas isso exigirá bastante trabalho. No exemplo abaixo, o elemento nav começa a se quebrar na medida em que a tela fica menor. Tudo se resume ao que funciona para o seu conteúdo.

<div class="container">
<nav>
  • Início
  • Taco Menu
  • Rascunhos
  • Horas
  • Direções
  • Contato
</nav>
<section>

Quando esse layout é muito estreito, o nav fica esmagado. Pior, não podemos utilizar min-width para corrigir isso, pois a coluna à direita não respeitaria isso.

</section>
