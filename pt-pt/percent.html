sobre este site

Percentagem é uma unidade de medida relativa ao bloco recipiente. É bom para imagens: aqui fazemos com que uma imagem seja sempre 50% da largura do seu recipiente. Tenta diminuir a página para ver o que acontece!

<article class="clearfix">

Podes até utilizar min-width e max-width para limitar o quanto a imagem pode ficar grande ou pequena!

</article>

layout com percentagem de largura

Podes utilizar percentagem para o layout, mas isto requer mais trabalho. No exemplo seguinte, o conteudo nav começa a envolver de uma forma desagradável quando a janela é demasiado estreita. Resume-se ao que funciona para o teu conteúdo.

<div class="container">
<nav>
  • Ínicio
  • Menu Taco
  • Lista Draft
  • Horas
  • Direcções
  • Contacto
</nav>
<section>

Quando este layout é demasiado estreito, a nav fica apertada. Pior, não podes utilizar min-width na nav para corrigir, porque a coluna da direita não o irá respeitar.
