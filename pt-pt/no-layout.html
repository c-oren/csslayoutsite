sobre este site

Não ter qualquer tipo de layout é quase ok se tudo o que queres é uma grande coluna de conteudo. No entanto, se o utilizado tornar a janela do browser realmente larga, torna-se um pouco irritante de ler: após cada linha os teus olhos de viajar uma longa distância da direita para a esquerda para a próxima linha. Tenta Tenta mudar o tamanho do teu browser para entenderes o que digo!

Antes de arranjarmos este problema, vamos ter a certeza que estamos entendidos numa importante propriedade de display.
