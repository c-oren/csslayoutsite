Il "Responsive Design" è la strategia secondo cui si realizza un sito web che "risponda" al browser e al dispositivo sul quale è visualizzato... cercando di mantenerlo gradevole in qualsiasi situazione.

Le media query sono lo strumento più potente per farlo. Prendiamo il nostro layout che usa le larghezze in percentuale e facciamo che sia visualizzato ad una sola colonna quando il browser è troppo piccolo per contenere il menù della barra laterale:
