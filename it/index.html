Questo sito insegna le basi CSS utilizzate per l'impaginazione di qualsiasi sito web.

Suppongo che tu sappia già cosa siano i selettori, le proprietà e i valori. E probabilmente sai già qualcosa sull'impaginazione, sebbene sia un'attività che ti fa rodere il fegato. Se vuoi imparare HTML e CSS da zero, dovresti consultare questo tutorial. Altrimenti, vediamo se possiamo risparmiarti un po' di nervosismo nel tuo prossimo progetto.
